College Information Management System (CIMS)

The College Information Management System (CIMS) is a web-based application developed as an academic capstone project to digitally manage college-level academic and administrative activities. The system simplifies the management of students, faculty, departments, classes, subjects, and examination results through a secure, role-based interface.

This project aims to reduce manual paperwork, improve data accuracy, and provide centralized access to academic information.
